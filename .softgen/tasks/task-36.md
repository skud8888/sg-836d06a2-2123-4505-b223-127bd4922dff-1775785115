## Notes
Create a floating feedback widget allowing users to submit bug reports, feature requests, and general feedback. Include screenshot capture, rating system, email notification to admins, and feedback management dashboard.

Features:
- Floating feedback button (bottom-right corner)
- Feedback form with categories (bug, feature, feedback)
- Rating system (1-5 stars)
- Screenshot capture functionality
- Email field pre-filled for logged-in users
- File attachment support
- Admin feedback dashboard
- Email notifications to admin team
- Feedback status tracking (new, in progress, resolved)
- User feedback history
