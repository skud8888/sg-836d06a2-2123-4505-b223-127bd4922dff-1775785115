# Admin Onboarding

## 🎯 Your First Hour as Admin

### **Step 1: Initial Setup (10 min)**

**1.1 Login to Admin Portal**
- Visit: `https://your-domain.com/admin/login`
- Use your admin credentials
- Enable 2FA if prompted (recommended)

**1.2 Complete Your Profile**
- Navigate to: Settings → Profile
- Upload profile photo
- Add contact information
- Set notification preferences

**1.3 System Health Check**
- Go to: System Health (`/admin/system-health`)
- Verify all systems show "Healthy" (green)
- Review any warnings or alerts
- Bookmark this page for daily checks

---

### **Step 2: Configure Organization Settings (15 min)**

**2.1 Company Information**
- Navigate to: Settings → Organization
- Add company name, logo, contact details
- Set business hours and timezone
- Configure email templates

**2.2 Course Categories**
- Go to: Course Management → Categories
- Create course categories (e.g., "Safety Training", "Technical Skills")
- Set category descriptions and icons

**2.3 Team Setup**
- Navigate to: Team Management (`/admin/team`)
- Invite team members (admins, trainers)
- Assign roles and permissions
- Send welcome emails

---

### **Step 3: Create Your First Course (20 min)**

**3.1 Course Template**
- Go to: Courses → Create New
- Fill in course details:
  * Course name
  * Description
  * Duration (hours)
  * Max students
  * Prerequisites
  * Price (if applicable)

**3.2 Course Content**
- Add course modules/sections
- Upload training materials (PDFs, videos)
- Create quizzes/assessments
- Set completion criteria

**3.3 Schedule First Class**
- Navigate to: Calendar (`/admin/calendar`)
- Click "New Event"
- Select course, date, time, location
- Assign trainer
- Set capacity
- Publish

---

### **Step 4: Student Enrollment (10 min)**

**4.1 Manual Enrollment**
- Go to: Students → Add Student
- Enter student details
- Select course and class
- Send enrollment confirmation

**4.2 Self-Enrollment Setup**
- Navigate to: Courses → [Course] → Settings
- Enable "Allow Self-Enrollment"
- Set enrollment requirements
- Configure waiting list (if needed)

---

### **Step 5: Learn the Dashboard (5 min)**

**Key Sections to Bookmark:**

1. **Dashboard** (`/admin`)
   - Quick stats and overview
   - Recent activity
   - Pending actions

2. **Bookings** (`/admin/bookings`)
   - Manage all enrollments
   - Process payments
   - Handle cancellations

3. **Calendar** (`/admin/calendar`)
   - View all scheduled classes
   - Create new sessions
   - Assign trainers

4. **Analytics** (`/admin/analytics`)
   - Student engagement
   - Course performance
   - Financial reports

5. **Documents** (`/admin/reports/documents`)
   - Student documents
   - Certificates
   - Compliance reports

6. **Audit Logs** (`/admin/audit-logs`)
   - Security monitoring
   - User activity tracking
   - System changes

## 🚨 Common Admin Scenarios

### **Scenario 1: Student Requests Certificate**
1. Go to: Students → [Student Name]
2. Click "Enrollments" tab
3. Find completed course
4. Click "Generate Certificate"
5. Review and send

### **Scenario 2: Cancel a Class**
1. Navigate to: Calendar
2. Click on the class
3. Select "Cancel Class"
4. Choose reason
5. System auto-notifies students

### **Scenario 3: Refund a Booking**
1. Go to: Payments → [Transaction]
2. Click "Refund"
3. Enter amount (full/partial)
4. Add reason
5. Confirm - funds returned in 5-7 days

### **Scenario 4: Add a New Trainer**
1. Navigate to: Team → Add Member
2. Select "Trainer" role
3. Enter details and credentials
4. Assign courses they can teach
5. Send invitation email

# Troubleshooting

## 🚨 Common Issues & Fixes

### **Issue: Can't Login**

**Possible Causes:**
- Incorrect credentials
- Account not activated
- Browser cache issues

**Solutions:**
1. Reset password using "Forgot Password"
2. Check email for activation link
3. Clear browser cache/cookies
4. Try incognito/private window
5. Contact support if persists

---

### **Issue: Course Not Showing**

**Possible Causes:**
- Course not published
- Prerequisites not met
- Class full

**Solutions:**
1. Check course availability dates
2. Verify prerequisites completed
3. Contact admin for enrollment
4. Join waiting list

---

### **Issue: Certificate Not Generated**

**Possible Causes:**
- Course not marked complete
- Assessment not passed
- System processing delay

**Solutions:**
1. Check course completion status
2. Verify all assessments passed
3. Wait 24 hours for processing
4. Contact admin if not received

---

### **Issue: Payment Failed**

**Possible Causes:**
- Insufficient funds
- Card declined
- Wrong card details

**Solutions:**
1. Verify card details correct
2. Check with bank
3. Try different payment method
4. Contact support for manual payment

---

### **Issue: Upload Failed**

**Possible Causes:**
- File too large (>10MB)
- Wrong file format
- Network issues

**Solutions:**
1. Compress file (use PDF/JPG)
2. Check file size < 10MB
3. Try again on stable network
4. Use mobile app for uploads

---

### **Issue: Notifications Not Received**

**Possible Causes:**
- Preferences disabled
- Email in spam
- Wrong contact info

**Solutions:**
1. Check notification settings
2. Verify email address correct
3. Check spam/junk folder
4. Add sender to contacts
5. Update preferences
